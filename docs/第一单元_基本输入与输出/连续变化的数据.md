# 《连续变化的数据》

> 🧰 创趣星盒板载显示屏、声音检测传感器。

<br><br>

功能： 屏显声强数值并绘制图形，动态表现声响变化的过程。

① 屏幕的尺寸为320x240像素，避免绘制的柱形超过最大屏幕高度240；

② 绘制与背景填充色相同的白色矩形用作每次的刷新处理；

③ 反应声强变化的柱形，起始点、宽度固定，高度根据声强变化而变化。

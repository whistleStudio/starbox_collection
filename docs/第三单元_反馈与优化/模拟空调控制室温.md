# 《模拟空调控制室温过程*》

> 🧰 创趣星盒板载温湿度传感器、按键、显示屏；外接风扇模块(P1)

<br><br>

功能：

屏幕显示当前环境温度和设定阈值。

当温度超过阈值时，风扇转动；否则，停止。

上、下按键分别控制加、减温度阈值。

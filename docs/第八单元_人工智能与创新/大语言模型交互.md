# 《大语言模型交互》

创趣星盒具有同大语言模型交互的能力，可运用于诸如文本生成、语言理解、知识问答、逻辑推理等场景。

> 🧰  创趣星盒板载麦克风、语音合成模块、WIFI-BLE模块和显示屏。

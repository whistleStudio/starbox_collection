# 《安全带未系提醒*》

> 🧰 创趣星盒板载按键、无源蜂鸣器和显示屏，外接称重传感器。

功能：

用称重传感器模拟汽车座椅的压力系统，左键按下/松开分别表示安全带的系上/松开状态；

称重传感器检测数值超过阈值且按钮松开时，蜂鸣器发出警报。
